
# 头部组件开发

### 初始化样式等参数

目录结构功能如下:
```
src
    common
        fonts--
    stylus
        index-----索引目录
        base------基础样式
        icon------icon标签样式
        mixin-----导入cub-ui下的mixin
        variable--导入cub-ui下的variable,自定义变量样式
    components
        support-ico---特价等小图标
```

##### components下的各个组件

**support-ico**
